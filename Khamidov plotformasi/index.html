JavaScript

  • JavaScript tarixi va imkoniyatlari

    JavaScript dasturlash tili 1995-yilda Brendan Eich tomonidan yaratilgan. Maqsad Netscape kompaniyasiga interaktiv imkoniyatlarni qo'shish uchun yaratilgan. JS birinchi "Mocha" deb atalgan. Keyinchalik "LiveScript"ga o'zgartirilgan. Va nihoyat Sun Microsystems kompaniyasi Java bilan hamkorligi tufayli bu tilni "JavaScript" deb nomladi.

    1-rasm
    2-rasm

  • Var, const, let

    var ➜ global o'zgaruvchi hisoblanadi (function scope)

    const ➜ o'zgarmas o'zgaruvchilar e'lon qilish

    let ➜ o'zgaruvchan o'zgaruvchi e'lon qilish (block scope)

  • Ma'lumot turlari (Data Types)

    JavaScriptda bir nechta turdagi ma'lumotlar mavjud:

    • String - matnli ma'lumotlar

    • Number - raqamli ma'lumotlar

    • Boolean - mantiqiy qiymatlar (true/false)

    • Object - ob'ektlar

    • Array - massivlar

    • Undefined va Null

    1-rasm

  • Amaliyot

    0. "Salom Dunyo" so'zini console.log ga chiqarish

    1. O'zgaruvchilarni e'lon qilish va barcha matematik amallarni bajarish

    2. Person ob'ektini yaratish

    3. Salom beradigan funksiya yaratish

    4. Talaba ob'ektini yaratish

    5. Talaba ob'ektini kalitlari yordamida qiymatlarini chiqarish

    6. Talaba ob'ektidagi qiymatlarni o'zgartirish

    7. Ikki sonni qo'shadigan funksiya yaratish

    8. Mevalar massivini yaratish

    9. Mevalar massividan ba'zi elementlarni olish

    10. Sonning kvadratini hisoblaydigan funksiya yaratish

  • Uyga vazifa

    1. O'z ismingizni console.log ga chiqarish

    2. User ob'ektini yaratish

    3. Funksiya chaqirilganda ismingizni qaytarsin

    4. 3 ta sonni qo'shadigan funksiya yozish

    5. Funksiyaga ism parametrini berish va o'zgartirish

    6. Aylana uzunligini hisoblovchi funksiya yozish

    7. Ikki sonni ko'paytiradigan funksiya yozish

    8. Person massivini yaratish

    9. Mashinalar massivini yaratish

    10. Mashinalar massividan ma'lum elementlarni olish

    11. Sonning kubini hisoblovchi funksiya yozish

    12. O'zgaruvchilarni e'lon qilish va konsolga chiqarish: ism, yosh, shahar o'zgaruvchilarini yaratib, console.log() yordamida chiqarish

    13. Matematik amallar: x va y o'zgaruvchilari bilan yig'indi, ayirma, ko'paytma va bo'linma hisoblab natijalarni chiqarish

    14. Ob'ektlar massivi: bir nechta talaba ob'ektlarini saqlash uchun "talabalar" massivini yaratish
